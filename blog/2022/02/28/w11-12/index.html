Report Viewing System

The report viewing system has been implemented on both frontend and backend. Here, each time a non admin users views an equipment, a viewing is added to the counter for that day. This means that views are totalled by usergroups for each day, and can be viewed as a total across a given date range.

Report Equipment Issue System

Along with developing the Equipment View page, the issue system has been implemented. On every Equipment page, there is a “Report Issue” button in which users can click to bring up a window. Here they input what issue they have with the equipment is. Admins can view a table pertaining to these issues. They can then click the solved checkbox when they have solved it.

Equipment Resource View and Editing

We are coming to the completion of our equipment resource editor. Here users can upload different types of media for the resources they wish to include onto the page.
