FAQs

Our client had asked us to include an FAQ section within the website for ease of use of all users - from admins to users. The FAQs also needed to be editable by admins only. In previous version, the FAQ section was a small question mark on our home page. With recent development, this can be accessed from the menu bar and from the user home page for all users.

Refactoring Code

As a team, we felt as if this was the best time to start refactoring code. Interaction between the client and server could have been tidier, and with deployment being a next step, it was crucial we could navigate through our code to fix bugs and ensure smooth deployment. We used environment variables with React in order to subsitute all calling to the backend in one place. We also tidied up our axios functions and created more components to be reused throughout the solution. This was also a great time to plan the documentation of our code, and implementing user guides as desired by our client.

Deployment

To deploy our code, we decided to use an Azure VM with Ubuntu. Having a client and server running on two different ports, deploying was not as straightforward as planned, and we now need to find a way that deployment can occur on one server to ease cost on hosting.
