System Design

System Architecture

The System Architecture diagram gives an overview on the key components of the solution. We have our Frontend section, which uses React to serve the frontend and handle navigation to views. The Backend section utilises Spring Boot, running a Tomcat server and using a REST API to controller different requests and CRUD actions. The Database layer uses PostgreSQL.
